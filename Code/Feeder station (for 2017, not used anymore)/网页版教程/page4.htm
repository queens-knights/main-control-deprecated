购买AU路由器，请点击。。。

 

打开管理页面：点我（如需用户名和密码请两个都输入admin）

 

看见如下界面：

 

2.点击无线网络设置->wlan2(2.4G)->基本设置：

3.在ssid一栏修改成你喜欢的无线网名称

点击应用设置，如果你还想继续设置请点击稍后重启，否则请点击立即重启

-----------------------------------------------------------------------------------------------------------------------------------------------

4.修改&设置无线网密码：点击无线网络设置->wlan2(2.4G)->无线安全：

加密类型改为WPA2，预共享密钥中填入你想要设置的无线网密码。

然后点击应用设置，如果你还想继续修改相关设置请点击稍后重启，否则请点击立即重启
