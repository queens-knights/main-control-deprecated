打开管理页面

1.

进入这个界面了（这是中文的页面截图，大致步骤就这样将就看吧）

 

2.

进入无线网络模式->wlan2(2.4G)基本设置

 

修改模式为Client,其它照着下图改（中继的信号信道必须在1-11不能在12和13信道）

3.

保存后不要重启进入无线安全：

Root-Client-XXXX就是你要中继的信号名，下面填你中继信号的密码和加密类型

再选择ssid下拉：中继AP-XXXX是你要重新发射出来的wifi信号，加密什么的根据自己需求来

应用，稍后重启。（设置好client后也可以用信号扫描来选择中继的信号）

7.

工作模式选择无线ISP

广域网接口选择问题：

中继2.4G信号，无线模式为2.4G 5G共存请选择：WLAN2

中继5G信号，无线模式为2.4G 5G共存请选择：WLAN1

无线模式为单一频率的请选wlan1,不行再试试wlan2

单一频率没有广域网接口选项！

应用后重启

 

 

8.

按下图设置无线高级（有一项不能选的忽略掉，2.4G和5G相同）

9.必须禁掉wps功能（2.4G和5G全部禁掉！），重启

 

 

重要说明：拜托各位第8步，第9步一定要严格按要求做！否则一定会出现过一段时间自动

 

重启掉线问题。
