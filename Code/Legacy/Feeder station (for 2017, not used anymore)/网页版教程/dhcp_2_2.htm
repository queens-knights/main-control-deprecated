问: 我电脑连接到路由器为什么才65M，与300M相差甚远。。

答: HOME SPOT这款路由器官方说明是支持300M连接速度的，但在很多情况下，电脑连接无线路由后显示的连接速度却是72M或者130M或者是150M，这属于正常的现象，因为不同无线芯片厂家之间还存在一定的兼容性，还有跟你附近的无线频段占用度，无线加密等等有很多大关系。

所以。你要是非常在意连接速度，我们并不建议你购买本产品，因为这个东西我们很难控制的。

但你可以从以下方面作调整，以达到更高的连接速度。

 

1.无线加密，有些电脑需要设置到WPA2 速度最高， 有些电脑需要设置到WPA/WPA2 mixed mode

2.更新无线网卡的驱动，让其处于最新驱动。

 

我们有2台笔记本测试情况如下

第一台 X230 笔记本自带150M 单频网卡

当路由器加密设置成WPA2时 连接速度是 150M

当路由器加密设置成WPA/WPA2 mixed mode时 连接速度是 150M

 

第二台 T430 笔记本自带300M双频无线网卡

当路由器加密设置成WPA2时 连接速度是 72M

当路由器加密设置成WPA/WPA2 mixed mode时 连接速度是 150M
