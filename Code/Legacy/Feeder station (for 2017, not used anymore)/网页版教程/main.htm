本路由器适合多种网络环境！所以首先要搞清楚自己是何种网络形式！搞清楚了才能比较快速的进行设置.如果您不清楚你的网络接入方式是那种！可参考以下类型！在此说明下！你是什么网络就看什么教程！对号入座！不是所有的教程你都需要看的！

 

以下ABCD 4种方式，你只需要看你对应的一种。不是4个都看啊。。。

 

A.你若是将本产品接入到现有的路由器后面，请将AP开关拨动到AP档（无线发射点模式）

这种网络形式常见局域网或者原本就有路由器的客人。

 

B．若您电脑直接接上ADSL Modem 时，需要设定ADSL帐户/密码后，而且需要进行连接动作后才可以连接至网络的话，请选择PPPoE

这种网络形式常见于ADSL宽带用户，小区宽带用户。也是国内用的最多的1种。

 

C．若您电脑直接接上ADSL或Cable Modem时，不需要做任何连接相关的设置即可连至网络的话，请选择DHCP Connection（动态IP）

这种网络形式常见局域网或者原本就有路由器的客人。

 

 

D．若您电脑直接接上ADSL或Cable Modem时，需要另外指定一个固定的IP地址后才可以连接至网络的话，请选择Static IP Address Connection（固定IP）。

这种网络形式常见于公司局域网，小区宽带用户。
